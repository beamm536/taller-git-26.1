Comandos Esenciales

Aquí encontrarás los comandos más utilizados en el día a día con Git:

Comando Descripción Ejemplo
git init Inicializa un repositorio local git init mi-proyecto
git clone Descarga un repositorio remoto git clone https://github.com/usuario/repo.git
git status Muestra el estado de los archivos git status
git add Agrega archivos al área de preparación git add . o git add archivo.txt
git commit Guarda los cambios con un mensaje git commit -m "Mensaje descriptivo"
git push Envía cambios al repositorio remoto git push origin main
git pull Descarga y fusiona cambios del remoto git pull origin main
git branch Lista o crea ramas git branch nueva-rama
git merge Fusiona una rama en la actual git merge nueva-rama

Introducción

Git es un sistema de control de versiones distribuido. Permite llevar un registro ordenado de los cambios en archivos.

Comandos Básicos

Comando Descripción
git status Visualizar estado de archivos
git add . Agregar archivos al staging
git commit -m "msg" Crear commit con mensaje
git push Enviar cambios al remoto

Flujo de Trabajo

  1. Visualizar estado: git status
  2. Preparar cambios: git add .
  3. Crear commit: git commit -m "mensaje"
  4. Enviar al remoto: git push
